安装maven

# windows开发环境安装mavne3.6+

因为必要的调试、开发，所以本地开发环境需要安装mavne，以便于更好的运行后端项目

# 下载mavne的安装包

下载地址：

maven官网 (opens new window)

请下载windows安装包

# 安装maven

  • 下载完成后将压缩包进行解压，会得到下面的结构：

  • 新建一个文件夹“repo”作为maven的仓库存储位置。

  • 然后进入maven安装目录，打开conf文件夹，将settings.xml文件，将 选项替换为刚刚创建的repo文件夹的路径即可，例如：

<localRepository>D:\maven\repo</localRepository>
1
  • 如果有需要，就将其配置为环境变量。
