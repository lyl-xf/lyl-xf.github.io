# 摄像头管理

# zlm平台列表

  • 这里的列表来自于配置文件中的zlm的配置信息。 这里的流媒体使用的是ZLMediaKit，链接地址为：https://github.com/ZLMediaKit/ZLMediaKit
  • 具体安装操作方式可以按照官网的操作方式进行安装。安装完成之后将配置信息维护到配置文件中
zlm:
  enable: true # 是否启用，未启用不会加载
  balance: round_robin # 节点负载均衡算法，默认round_robin
  nodes: 
    - server-id: zlm-node-1 # 节点ID，可以自定义名称
      host: "127.0.0.1" # 节点地址
      secret: KqErX1KUSnoGn3IsdsdspHl8KyLSAdmVgLlpa # 节点密钥 来自于zlm中的config.ini 中的 secret，如果不一致则无法进行通讯。
      enabled: true # 节点是否启用
      hook-enabled: true # 节点是否启用hook接口，启用hook会注入hook接口，默认true，需要注意拦截器放通
