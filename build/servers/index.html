简单部署

# 简单部署

这里仅仅介绍最简单的部署方式，如果喜欢用Jenkins，或者是docker来部署，请自行修改。

# 后端打包

1、检查当前分支，以及配置是否正确

2、使用mavn打包

3、将打好的包上传至服务端（打好的包在：sana-server/target/sana-server.jar）

4、启动服务端

# 前端打包

1、修改配置：sana-web/.env.production

# 生产环境
NODE_ENV = production

# 标题
VUE_APP_TITLE = SANA

# 接口地址
VUE_APP_API_BASEURL = http://生产环境ip:端口/sana

# 本地端口
VUE_APP_PORT = 2800

# 是否开启代理
VUE_APP_PROXY = true
