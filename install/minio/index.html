安装MinIO

# 安装minio

这里建议mysql安装在服务端，不在本地安装。为了方便，这里使用docker进行安装

# docker安装minio，或者安装最新版也可以

docker run   -p 9000:9000 -p 9090:9090 --name=lana-minio -d  --restart=always -v /home/minio/data:/data -v /home/minio/config:/root/.minio  -e "MINIO_ACCESS_KEY=lana" -e "MINIO_SECRET_KEY=admin1234"  minio/minio:latest server /data --console-address ":9090" -address ":9000"
1

# 访问minio

访问地址格式：http://ip:9090

# 说明

这里只是一个推荐的安装方式，实际安装方式根据个人需求进行选择。
