# 找到sana-web

  • 使用编辑器将sana根目录下的sana-web项目拷贝到独立的文件下（请一定要将sana-web剪切、粘贴出来，单独存放。）。
  • 使用编辑器（vs code,或者webstrom）打开sana-web项目：

# 修改配置

  • 修改配置文件：sana-web/.env.development
  • 将 'VUE_APP_API_BASEURL' 修改为为自己的后端地址。

# 运行项目：

  • 编辑项目：npm install / yarn install

  • 启动项目：npm run serve / yarn serve
